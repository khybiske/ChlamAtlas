Mutants tab: list row design

Target gene(s) should be the most visually prominent element. Three options for how each row is laid out in the scrollable list panel.

A

Target gene(s) as headline

incA CTL0496 KO
YW082 · CT-L2
IV+ IO?
scc4 +3 others KO
KH001 · CT-L2 · CTL0397–399
IV+ IO+
CTL0530 Tn +372bp
YW030 · CT-L2
IV− IO?

Gene symbol (or locus tag if unnamed) is the visual anchor. Mutant ID + strain below. Type badge (KO, Tn, Chimera…) inline. Transposons show insertion offset. Pipeline dots right-aligned. Phenotype badges (IV+/−, IO+/−/?) far right.

B

Compact two-line rows with gene chip(s)

YW082 KO CT-L2
incA IV+ IO?
KH001 KO CT-L2
scc4 CTL0398 CTL0399 IV+ IO+
YW030 Tn CT-L2 · +372bp
CTL0530 IV− IO?

Mutant ID is the top line (useful for lab members), gene chip(s) on the second line as clickable tags. More compact — fits more mutants on screen. Trade-off: gene name is slightly less dominant than option A.

Key for both options — IV = in vitro phenotype, IO = in vivo phenotype, ? = untested.
Pipeline dots: Transform → Clone → Genotype → In vitro → In vivo → Sequenced.
